# Heat Pump Butler

A simple, one-page marketing website for a residential heat pump cleaning business.

## Live site

Published via GitHub Pages: (link added after first deploy)

## Structure

- `index.html` — the entire site (HTML + embedded CSS + a touch of JS), no build step required.

## Editing

Open `index.html` in any editor and update the placeholder business name, phone, email, and service area. Push to `main` and GitHub Pages redeploys automatically.
